# ForwardList

`ForwardList` — это реализация односвязного списка на C++. Класс предоставляет основные операции для работы со списком, такие как добавление, удаление, поиск и очистка элементов.

## Основные методы

### Конструкторы и деструкторы
- **`ForwardList()`**  
  Конструктор по умолчанию. Создает пустой список.

- **`ForwardList(const ForwardList& other)`**  
  Конструктор копирования. Создает копию списка `other`.

- **`ForwardList(ForwardList&& other)`**  
  Конструктор перемещения. Перемещает данные из списка `other` в текущий список.

- **`~ForwardList()`**  
  Деструктор. Освобождает память, занимаемую списком.

### Операторы
- **`ForwardList& operator=(const ForwardList& other)`**  
  Оператор присваивания копированием. Копирует данные из списка `other` в текущий список.

- **`ForwardList& operator=(ForwardList&& other)`**  
  Оператор присваивания перемещением. Перемещает данные из списка `other` в текущий список.

### Методы для работы со списком
- **`void push_front(double value)`**  
  Добавляет элемент `value` в начало списка.

- **`void pop_front()`**  
  Удаляет первый элемент списка. Если список пуст, выводит сообщение об ошибке.

- **`const double& front() const`**  
  Возвращает значение первого элемента списка. Если список пуст, выводит сообщение об ошибке.

- **`void display() const`**  
  Выводит все элементы списка в консоль.

- **`size_t size() const`**  
  Возвращает количество элементов в списке.

- **`bool empty() const`**  
  Проверяет, пуст ли список. Возвращает `true`, если список пуст, иначе `false`.

- **`void clear()`**  
  Очищает список, удаляя все элементы.

- **`void erase(double value)`**  
  Удаляет все элементы списка, равные `value`.

## Пример использования

```cpp
#include "headers.h"
#include <iostream>

int main() {
    ForwardList list;
    list.push_front(3.14);
    list.push_front(2.71);
    list.push_front(1.62);

    std::cout << "Список: ";
    list.display(); // Вывод: 1.62 2.71 3.14

    std::cout << "Размер списка: " << list.size() << std::endl; // Вывод: 3

    list.pop_front();
    std::cout << "Список после удаления первого элемента: ";
    list.display(); // Вывод: 2.71 3.14

    list.erase(2.71);
    std::cout << "Список после удаления элемента 2.71: ";
    list.display(); // Вывод: 3.14

    list.clear();
    std::cout << "Список после очистки: ";
    list.display(); // Вывод: (пусто)
    std::cout << "Список пуст? " << (list.empty() ? "Да" : "Нет") << std::endl; // Вывод: Да

    return 0;
}
